华为纵向合作项目 兼 四川智翎公司横向项目



项目背景: 针对智能座舱抬头显示 HUD 研发过程中，生理反馈数据（人因数据）与动态仿真场景脱节、设计方案到仿真环境部署链路过长的问题，开发一套集“多维硬件接入、算法耦合、动态 UI 渲染”于一体的闭环实验平台。

行动方案:
1 多源异构设备集成架构：基于 UDP 通讯协议，构建了兼容多家设备厂商（如眼动仪、脑电仪、生理监测仪等）的通用接入层，实现了人因数据在仿真环境中的毫秒级同步。
2 仿真引擎深度二次开发：针对 CARLA（基于 UE4）进行深度定制，利用 Python API 扩展底层功能，将外部人因特征量实时注入仿真世界，方便人因算法实时接入。
3 自适应 HUD 渲染子系统：开发基于 pygame 的高性能渲染引擎，支持AR导航信息、信号灯预警及目标检测 Bounding Box 的实时叠加显示。
4 设计驱动的工程流水线：打通Figma设计端到仿真端的链路，实现基于JSON配置文件的UI动态解析与布局更新，大幅缩短从UI设计到仿真实验的迭代周期。


项目成果：
1、建成国内领先的 HUD 人因工程仿真验证平台，实现眼动仪、脑电仪、生理监测仪等多源人因设备统一接入，支持毫秒级数据同步与实时注入自动驾驶仿真环境。
2、完成CARLA 仿真引擎深度定制，构建外部人因特征—仿真世界—HUD渲染的实时联动链路，为人因算法、感知算法、座舱交互算法提供标准化验证环境。
3、平台稳定支撑上百人次人因驾驶实验，覆盖高校、科研院所及企业研发团队，为 HUD 交互优化、自动驾驶人机共驾安全性研究提供关键数据与实验支撑。
4、平台功能模块正式接入公司 Ozone 数字孪生一站式自动驾驶仿真与测试平台（https://ozone.zhilingtech.com/），成为其人因研究、智能座舱仿真、Corner Case 安全测试的核心能力模块，完善数字孪生全栈测试能力。
